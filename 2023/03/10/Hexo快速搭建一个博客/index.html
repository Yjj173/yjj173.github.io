Hexo快速搭建一个博客

发表于2023-03-10|更新于2023-03-10
|阅读量:

前言

首先非常感谢羊哥的教程，跟着一步步操作，非常快速地搭建好了个人博客。这里抛出一个链接：
b站教学视频：https://www.bilibili.com/video/BV1Yb411a7ty

搭建博客

一 前期准备

①安装Node.js

官网链接 https://nodejs.org/en/

②为了解决速度慢的问题。建议给Node.js换一个镜像源，这里根据羊哥的教程更换了淘宝的镜像源，代码如下：

npm install -g cnpm –registry=https://registry.npm.taobao.org

③安装Hexo

官网链接：https://hexo.io/
直接在首页复制代码，粘贴运行就好：
cnpm install -g hexo-cli

快速搭建

①初始化

在本地创建一个文件夹，作为博客部署的空间。（注：以后所有的操作都基于此文件夹，如果出现了什么问题无法解决，删掉重来就好）在此文件夹内初始化hexo：

hexo init
exo init
接下来就会在此目录下自动生成一些文件，初始化完成。

hexo s

通过本地默认的4000端口访问：http://localhost:4000

清理缓存 hexo clean
生成静态文件 hexo g

②部署到github

①到github去新建仓库

注意命名格式：账号名.github.io（例：CNhuazhu.github.io）

②安装git部署的插件
cnpm install –save hexo-deployer-git

③设置_config.yml文件

在博客目录下找到_config.yml文件，最底部有一个# Deployment模块，在此设置：

deploy：
type: git
repo: https://github.com/Yjj173/yjj173.github.io.git
branch:master

④部署到远端
hexo d，跳出弹框，输入找你自己的github账号与邮箱
后在命令行中输入 hexo d

总结

这样就部署完毕了，可以通过输入你github对应的网址来通过游览器查看网页。
例如，我通过yjj173.github.io来进行访问。
